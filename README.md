# Godot_project_prototype
2025-2학기 Creative Media Programming 수업 Team 6의 기말 Godot 프로젝트입니다.


GameManager.gd의 is_cheat bool로 치트 상태 조정가능
GameManager는 오토로드에 등록해놓았으니까 다른 곳에서도 쓸 수 있습니다. 
예. 치트인지 확인하고 싶을때 어디서나 if GameManager.is_cheat: 로 확인 가능

-------------- 치트 기능(F1로 치트 토글 가능) ---------------
F2 : 화면 멈추기 - 멈춘 후 마우스 드래그로 카메라 이동 가능
K : 스킬 장착 / 강화 / 합성 탭 열기
G : 보상 창 열기a
